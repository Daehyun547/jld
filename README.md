## 1일차: 변산반도-광주

|시간|내용|
|-|-|
|10:00-12:00|대전→변산반도|
|12:00-13:00|점심 (백합식당)|
|13:00-15:00|채석강|
|15:15-15:45|(선택) 적벽강, 슬지제빵소/풍요일지/마르|
|16:15-17:30|내소사, (선택) 개암사|
|18:00-19:00|저녁 (곰소 고향맛횟집/현정이네)|

|20:00-|에이치원호텔|

1. 채석강 (바다 + 절, 대비감)
   - 간조 전후 필수: [물때 확인](https://m.badatime.com/237.html)
     - 12/28(일) 14:36 간조 → 13:36~15:36
   - 수평층 암벽 + 파도 → 사진 임팩트 큼
   - 날 맑으면 여기서 이미 “여행값” 함
   - 짧은 트레킹 (해안 암반 산책): 30~60분
2. (선택) 적벽강
   - 채석강보다 사람 적고
   - 붉은 절벽 + 바다 대비 좋음
3. 변산 드라이브
   - 격포항 → 곰소염전 방향
   - 길이: 약 15~20분
   - 바다·갯벌·염전 풍경 변화 있음
   - 차로 이동 → 내려서 감상
4. 내소사
   - 전나무숲 → 절 → 계곡
   - 바다 다음에 오면 대비가 아주 좋음
5. (선택) 개암사

   - 오전 방문 추천 (사람 적고 분위기 최고)
   - 소요: 40~60분
2. 죽녹원
   - 소요: 60~90분
   - 오전 방문 추천 (사람 적고 사진 깔끔)
   - 겨울에도 대숲은 색감 유지됨
   - 걷는 난이도 낮음 → 장거리 운전 전 딱 좋음
3. 관방제림
   - 죽녹원 바로 옆
   - 소요: 20~30분
   - 강변 + 고목 → 시야 확 트임
4. (선택) 메타프로방스
5. (선택) 실상사
   - 지리산 자락의 고요한 분위기
   - 관광지 느낌 거의 없음 → 여행 템포가 내려감
6. (선택) 만복사지
7. 광한루원
    - 정자·연못·누각 → 안정적인 사진

## 3일차: 구례

|시간|내용|
|-|-|
|8:30-9:00|남원→구례|
|9:00-10:30|화엄사|
|10:45-11:25|천은사|
|11:40-12:40|점심|
|13:00-14:30|사성암|
|15:10-16:10|(선택) 피아골|
|16:10-16:50|(선택) 연곡사|
||저녁|

1. 화엄사
   - 오전 방문 필수
   - 대가람 + 지리산 배경 → 스케일감 최고
2. 천은사
   - 화엄사와 대비되는 소박함
   - 물·숲·절의 균형
3. 섬진강 상류 드라이브 (화엄사 ↔ 구례읍 사이)
   - 하동 쪽 안 내려가도 충분
   - 강·산·논이 동시에 보이는 구간 (차 안에서 봐도 풍경이 완성)
4. 구례읍 식사
   - 이 날은 맛집 몰아서 먹어도 되는 날
   - 촌닭구이 맛나식당(예약필수), 당골식당
   - 5일장터안에 수구레국밥, 가마솥소머리국밥, 백련산방 한정식, 청년푸드마켓의 조던해장국
   - 구례읍 안에는 갯바위 해산물 오마카세, 월성가든 육회비빔밥, 순대백화점 파김치, 양진맛집 생선구이정식, 선미옥 다슬기 수제비,
   - 구례구역 앞에는 미광가든 참게탕, 구례대합실 다슬기 파스타
   - 화엄사 앞에는 예원, 만남가든, 할팽정조
5. 사성암
   - 구례 사진의 끝판왕
   - 절 + 전망대 + 암자 조합
6. (선택) 섬진강 대나무숲길
   - 평지, 짧은 산책
   - 바람만 덜 불면 겨울에도 괜찮음
   - 사진보다는 리듬 조절용
7. (선택) 피아골
   - 지리산 계곡 중 접근성·완성도 최고
   - 겨울에도 물·바위·숲 구조감이 살아 있음
   - 차 세우고 짧은 산책만 해도 충분
8. (선택) 연곡사
   - 피아골 안에 함께 묶기 좋음
   - 화엄사·천은사와 성격이 다름
   - 관광지 느낌 거의 없음
9. (선택) 지리산 성삼재 (공사), 노고단, 쌍산재 (휴무), 구만리 저수지 캠핑장, 양떼 농장, 지리산 수목원, 지리산 스카이런, 쌍계사, 섬진강 어류생태관, 압화 박물관, 고소성 등
