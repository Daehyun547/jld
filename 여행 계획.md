# 1일차: 변산반도
1. 이른 점심 (11시쯤?)
2. 채석강 (바다 + 절, 대비감)
   - 간조 전후 필수: [물때 확인](https://m.badatime.com/237.html)
     - 12/27(토) 13:33 간조 → 12:33~14:33
     - 12/28(일) 14:36 간조 → 13:36~15:36
   - 수평층 암벽 + 파도 → 사진 임팩트 큼
   - 날 맑으면 여기서 이미 “여행값” 함
   - 짧은 트레킹 (해안 암반 산책): 30~60분
3. 적벽강
   - 채석강보다 사람 적고
   - 붉은 절벽 + 바다 대비 좋음
4. 변산 드라이브
   - 격포항 → 곰소염전 방향
   - 길이: 약 15~20분
   - 바다·갯벌·염전 풍경 변화 있음
   - 차로 이동 → 내려서 감상
5. 내소사
   - 전나무숲 → 절 → 계곡
   - 바다 다음에 오면 대비가 아주 좋음
6. (선택) 개암사

# 2일차: 남원 (역사 + 정리의 날)
1.  광한루원
    - 오전 추천 (사람 적음)
    - 정자·연못·누각 → 안정적인 사진
2. 실상사
   - 지리산 자락의 고요한 분위기
   - 관광지 느낌 거의 없음 → 여행 템포가 내려감
3. 남원 시내 산책 + 식사
   - 이 날은 일부러 일찍 끝내는 게 좋음
   - 다음 날(구례)을 위해 체력 비축
4. (선택) 만복사지

# 3일차: 구례 (여행의 하이라이트)
1. 화엄사
   - 오전 방문 필수
   - 대가람 + 지리산 배경 → 스케일감 최고
2. 사성암
   - 구례 사진의 끝판왕
   - 절 + 전망대 + 암자 조합
3. 섬진강 상류 드라이브
   - 하동 쪽 안 내려가도 충분
   - 강·산·논이 동시에 보이는 구간
4. 구례읍 식사
   - 이 날은 맛집 몰아서 먹어도 되는 날
   - 촌닭구이 맛나식당(예약필수), 당골식당
   - 5일장터안에 수구레국밥, 가마솥소머리국밥, 백련산방 한정식, 청년푸드마켓의 조던해장국
   - 구례읍 안에는 갯바위 해산물 오마카세, 월성가든 육회비빔밥, 순대백화점 파김치, 양진맛집 생선구이정식, 선미옥 다슬기 수제비,
   - 구례구역 앞에는 미광가든 참게탕, 구례대합실 다슬기 파스타
   - 화엄사 앞에는 예원, 만남가든, 할팽정조
5. (선택) 지리산 성삼재, 노고단, 천은사, 쌍산재, 섬진강 대나무 숲길, 구만리 저수지 캠핑장, 양떼 농장, 지리산 수목원, 지리산 스카이런, 피아골, 연곡사, 쌍계사, 섬진강 어류생태관, 압화 박물관, 고소성 등
