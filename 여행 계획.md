# 1일차: 변산반도
1. 채석강 (바다 + 절, 대비감)
   - 오후~해질녘 추천
   - 수평층 암벽 + 파도 → 사진 임팩트 큼
   - 날 맑으면 여기서 이미 “여행값” 함
2. 내소사
   - 전나무 숲길이 메인
   - 바다 다음에 오면 대비가 아주 좋음
3. 변산 해안도로 드라이브
   - 사진보다 차 안에서 보는 풍경이 좋은 구간

# 2일차: 남원 (역사 + 정리의 날)
1.  광한루원
    - 오전 추천 (사람 적음)
    - 정자·연못·누각 → 안정적인 사진
2. 실상사
   - 지리산 자락의 고요한 분위기
   - 관광지 느낌 거의 없음 → 여행 템포가 내려감
3. 남원 시내 산책 + 식사
   - 이 날은 일부러 일찍 끝내는 게 좋음
   - 다음 날(구례)을 위해 체력 비축

# 3일차: 구례 (여행의 하이라이트)
1. 화엄사
   - 오전 방문 필수
   - 대가람 + 지리산 배경 → 스케일감 최고
2. 사성암
   - 구례 사진의 끝판왕
   - 절 + 전망대 + 암자 조합
3. 섬진강 상류 드라이브
   - 하동 쪽 안 내려가도 충분
   - 강·산·논이 동시에 보이는 구간
4. 구례읍 식사
   - 이 날은 맛집 몰아서 먹어도 되는 날

# 4일차: 용궐산 (활동으로 마무리)
1. 용궐산
   - 왕복 1.5~2시간
   - 암릉 + 강 조망 → 사진도 좋고 성취감도 있음
2. (선택) 채계산 출렁다리
   - 체력 남으면
   - 용궐산보다 관광형
